# SWE301 Lab IV - Remote Health Monitoring System

A Java-based implementation of a remote health monitoring system for SWE301 Lab IV.

## Classes
- `Patient` - Patient information and management
- `Doctor` - Doctor information and patient assignment  
- `VitalSignRecord` - Vital signs measurement and tracking
- `WearableDevice` - Medical device simulation
- `MonitoringService` - Health monitoring and alert system
- `Alert` - Medical alert generation and management
- `Main` - System demonstration
